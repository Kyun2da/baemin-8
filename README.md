- VS code

vs code에서 `Command + Shift + P`를 누른 뒤 `Shell Command: Install 'code' command in PATH`를 실행하면 터미널에서 `code` 명령어를 통해 vs code를 실행할 수 있어요.  
뒤에 폴더명을 붙인다면 해당 폴더를 vs code 내에서 엽니다!
